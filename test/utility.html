# Display #
Type of display

Untuk mengatur display pada komponen di tata letak, class yang tersedia yaiu sebagai berikut.

Display juga mendukung penggunaan breakpoints dengan menyisipkan kode breakpoint pada tanda *.

Class Keterangan
d-none atau d-*-none Display None
d-block atau d-*-block Display Block
d-flex atau d-*-flex atau flex- Display FLex & Wrap
d-grid atau d-*-grid Display Grid
d-inline atau d-*-inline Display Inline
d-inline-block atau d-*-inline-block Display Inline Block
d-inline-flex atau d-*-inline-flex Display Inline Flex
d-inline-table atau d-*-inline-table Display Inline Table
# Shadows #
Type of shadow

Shadow memiliki 2 mode yaitu default dan dark dimana masing-masing memiliki 3 tipe yaitu default, small, dan large.

Shadow juga mendukung penggunaan breakpoints dengan menyisipkan kode breakpoint pada tanda *.

Shadow small shadow-small atau shadow-*-small
Only shadow shadow atau shadow-*
Shadow large shadow-large atau shadow-*-large
Shadow dark small shadow-dark-small atau shadow-*-dark-small
Only dark shadow shadow-dark atau shadow-*-dark
Shadow dark large shadow-dark-large atau shadow-*-dark-large
